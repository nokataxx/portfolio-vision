# portfolio-vision

株式ポートフォリオ モンテカルロシミュレーター — 過去の株価データをもとに、将来のポートフォリオ価値の確率分布を可視化するWebアプリケーションです。

## Tech Stack

- React 19 + TypeScript（Vite 7）
- Tailwind CSS v4
- shadcn/ui（radix-nova スタイル）
- react-router v7
- lucide-react

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| コマンド | 説明 |
|---|---|
| `npm run dev` | 開発サーバー起動 |
| `npm run build` | 型チェック + ビルド |
| `npm run lint` | ESLint 実行 |
| `npm run preview` | 本番ビルドのプレビュー |
