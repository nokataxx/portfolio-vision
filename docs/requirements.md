## 1. プロジェクト概要

### 1.1 背景・目的

個人投資家が保有する日本株ポートフォリオについて、過去の株価実績データをもとにモンテカルロシミュレーションを実施し、将来の評価額の確率分布を可視化するWebアプリケーションを開発する。

「このポートフォリオは10年後にどうなっているか」という問いに対して、単一の予測値ではなく確率分布として答えることで、リスクを定量的に把握した上での資産形成の意思決定を支援する。

### 1.2 対象ユーザー

- 日本株を保有する個人投資家
- ポートフォリオのリスク・リターン特性を把握したい方
- 長期投資の観点で将来の資産評価を確認したい方

### 1.3 提供価値

- 保有銘柄の過去株価データを自動取得し、統計量（平均リターン・標準偏差・相関）を算出
- モンテカルロシミュレーションによる将来評価額の確率分布の可視化
- 元本割れ確率・期待リターンなど直感的な指標の提示
- 銘柄ごとのリスク寄与度の把握

## 2. 機能要件

### 2.1 銘柄入力機能

ユーザーが保有銘柄の情報を入力する。複数銘柄の登録・編集・削除が可能。

| 入力項目 | 形式 | 例 | 説明 |
|---------|------|-----|------|
| 証券コード | 4桁数字 | 7203 | 東証上場銘柄コード |
| 銘柄名 | 自動取得 | トヨタ自動車 | 証券コード入力後に自動補完 |
| 保有株数 | 整数 | 100 | 単元株未満も入力可 |
| 取得単価 | 円 | 2,800 | 1株あたりの購入価格 |

操作仕様：
- 証券コードを入力すると銘柄名を自動取得・表示
- 銘柄の追加・削除はダイナミックに行える（ページ遷移なし）
- 最大保有銘柄数：20銘柄
- 入力データはブラウザのローカルストレージに保存し、次回アクセス時に復元

### 2.2 株価データ取得機能

#### 2.2.1 データソース

| データソース | 備考 |
|------------|------|
| Yahoo Finance API（非公式） | 無料・登録不要・日本株対応・前日終値まで取得可能 |

将来的な J-Quants API（JPX公式）への移行を考慮し、データ取得レイヤーはインターフェースで抽象化して実装する（4.2参照）。商用化の際は J-Quants API への移行を検討する。

#### 2.2.2 取得データ仕様

- 取得期間：直近5年間（デフォルト）／3年・10年を選択可能
- 取得頻度：日次終値
- データ項目：日付・始値・高値・安値・終値・出来高・調整後終値
- 調整後終値を使用（株式分割・配当の影響を除去）

#### 2.2.3 データ取得のタイミング

- 銘柄登録時に自動取得
- 「データ更新」ボタンで手動再取得可能
- 取得済みデータはセッション中キャッシュし、不要なAPIリクエストを抑制

### 2.3 統計量計算機能

取得した株価データから以下の統計量を算出する。

**個別銘柄ごと**
- 日次リターン（対数リターン）の時系列
- 年率平均リターン（μ）
- 年率標準偏差（σ）：ボラティリティ
- シャープレシオ（無リスク金利：0.1%想定）
- 最大ドローダウン

**ポートフォリオ全体**
- 銘柄間の相関係数行列
- ポートフォリオの年率期待リターン（加重平均）
- ポートフォリオの年率標準偏差（共分散行列を考慮）
- 各銘柄のリスク寄与度（%）

### 2.4 シミュレーション計算機能

#### 2.4.1 計算方式

Phase 1（シンプル版）とPhase 2（本格版）の2段階で実装する。

**Phase 1：各銘柄独立モデル**
- 銘柄間の相関を無視し、各銘柄を独立したGBM（幾何ブラウン運動）でシミュレート
- 計算が単純で実装・検証が容易

**Phase 2：相関考慮モデル**
- 共分散行列のコレスキー分解を使用
- 銘柄間の相関を反映した多変量正規分布から乱数を生成
- より現実的なシミュレーション

#### 2.4.2 シミュレーションパラメータ

| パラメータ | 単位 | デフォルト値 | 説明 |
|-----------|------|------------|------|
| シミュレーション期間 | 年 | 10 | 1〜30年で選択可 |
| シミュレーション回数 | 回 | 10,000 | モンテカルロ試行回数 |
| 年次追加投資額 | 万円 | 0 | 毎年ポートフォリオに追加投資する金額 |
| 無リスク金利 | % | 0.1 | シャープレシオ計算用 |

#### 2.4.3 計算処理

- Web Worker を使用しUIをブロックしない
- 進捗バーで計算進捗を表示
- 計算時間の目安：10,000回 × 20銘柄 × 10年 を5秒以内に完了

### 2.5 結果表示機能

#### 2.5.1 ポートフォリオサマリー

現在の状況を一覧表示する。

| 指標 | 説明 |
|------|------|
| 現在の評価額合計 | 前日終値ベースの時価総額 |
| 取得価格合計 | 入力した取得単価 × 株数の合計 |
| 含み損益 | 評価額 − 取得価格合計 |
| 含み損益率 | % |
| ポートフォリオ期待リターン | 年率 % |
| ポートフォリオボラティリティ | 年率 % |
| シャープレシオ | — |
| 過去最大ドローダウン | 取得期間中の最大下落率（%）|

> **リスク許容度について**
> 本アプリではリスク許容度の入力は求めない。代わりに過去最大ドローダウン・悲観シナリオ（10パーセンタイル）・元本割れ確率などの情報を提示し、ユーザー自身が「このリスクに耐えられるか」を判断できる材料を提供する。

#### 2.5.2 シミュレーション結果サマリーカード

| 指標 | 説明 |
|------|------|
| N年後 期待評価額（中央値） | 万円 |
| N年後 楽観シナリオ（90パーセンタイル） | 万円 |
| N年後 悲観シナリオ（10パーセンタイル） | 万円 |
| 元本割れ確率 | 取得価格を下回る確率（%） |
| 2倍達成確率 | 取得価格の2倍を超える確率（%） |

#### 2.5.3 グラフ表示

**グラフ①：将来評価額の確率分布（ヒストグラム）**
- N年後の評価額の度数分布を表示
- 10・50・90パーセンタイルのラインを表示
- 取得価格総額のラインを表示（元本割れの境界）

**グラフ②：評価額の年次推移（帯グラフ）**
- X軸：経過年数（0〜N年）
- Y軸：ポートフォリオ評価額（万円）
- 10〜90パーセンタイルの帯と中央値ラインを表示
- 取得価格総額のラインを重ねて表示

**グラフ③：元本割れ確率の推移**
- X軸：経過年数
- Y軸：元本割れ確率（%）
- リスクが時間とともにどう変化するかを可視化

**グラフ④：銘柄別リスク寄与度（円グラフ）**
- 各銘柄がポートフォリオ全体のリスクに占める割合を表示
- 銘柄の集中リスクの把握に使用

**グラフ⑤：銘柄別過去株価の推移（折れ線グラフ）**
- 取得期間中の各銘柄の株価推移
- 正規化表示（取得時点を100として指数化）で銘柄間の比較を容易にする

#### 2.5.4 銘柄別詳細テーブル

| 銘柄コード | 銘柄名 | 保有株数 | 取得単価 | 現在値 | 評価額 | 損益 | 損益率 | 年率リターン | ボラティリティ | リスク寄与度 |
|-----------|-------|---------|---------|-------|-------|------|-------|------------|-------------|-----------|

## 3. 非機能要件

### 3.1 パフォーマンス

| 項目 | 要件 |
|------|------|
| 株価データ取得 | 1銘柄あたり3秒以内 |
| シミュレーション実行 | 10,000回 × 20銘柄 × 10年を5秒以内 |
| グラフ描画 | 計算完了後1秒以内 |
| 初期表示 | 3秒以内（LCP） |

### 3.2 対応環境

| 項目 | 対応範囲 |
|------|---------|
| ブラウザ | Chrome / Firefox / Safari / Edge（最新2バージョン） |
| デバイス | PC・タブレット・スマートフォン（レスポンシブ対応） |
| 画面解像度 | 375px〜1920px |

### 3.3 ユーザビリティ

- 証券コード入力時の銘柄名オートコンプリート
- 各指標へのツールチップ説明
- ポートフォリオデータのJSON形式でのエクスポート・インポート
- シミュレーション結果のPNG/PDFエクスポート
- データ取得エラー時のわかりやすいエラーメッセージ表示

### 3.4 データの取り扱い

- 入力データはブラウザのローカルストレージのみに保存（サーバー送信なし）
- 株価データのキャッシュはセッションストレージに保存
- 個人情報・証券口座情報は一切取り扱わない

### 4.2 データソース切り替えへの備え

Yahoo Finance API から J-Quants API への移行を見越し、データ取得レイヤーをインターフェースで抽象化する。プロバイダーを切り替えるだけでデータソースを変更できる設計にしておく。

```typescript
interface StockDataProvider {
  fetchHistoricalPrices(code: string, from: Date, to: Date): Promise<PriceData[]>
  fetchStockInfo(code: string): Promise<StockInfo>
}

class YahooFinanceProvider implements StockDataProvider { ... }  // 現在使用
class JQuantsProvider implements StockDataProvider { ... }       // 商用化時に切り替え
```

**各プロバイダーの比較**

| 項目 | Yahoo Finance API | J-Quants API |
|------|-----------------|--------------|
| 費用 | 無料 | 無料〜16,500円/月 |
| 公式サポート | なし（非公式） | あり（JPX公式） |
| 直近データ | 前日終値まで | 無料プランは12週間遅延 / 有料プランはリアルタイム |
| 利用規約 | グレーゾーン（非商用のみ推奨） | 明確（商用利用可） |
| 安定性 | 仕様変更リスクあり | 高い |

## 7. 考慮事項・制約事項

### 7.1 計算上の簡略化

- 配当金・株主優待は評価額に含めない（株価の値動きのみを対象）
- 税金（譲渡所得税・配当税）は計算に含めない
- 売買手数料は考慮しない
- 上場廃止リスクは考慮しない
- 過去の統計量が将来も継続すると仮定（定常性の仮定）

### 7.2 データソースに関するリスク

Yahoo Finance API は公式サポートのない非公式APIであり、仕様変更・サービス変更により予告なく使用不可になる可能性がある。個人・非商用利用の範囲で使用する。商用化の際は J-Quants API（JPX公式）への移行を検討する。

### 7.3 免責事項

本ツールの計算結果はあくまで参考値であり、将来の株価・運用成果を保証するものではありません。投資判断はご自身の責任で行い、必要に応じて専門家への相談を推奨します。
